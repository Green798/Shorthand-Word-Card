# 速记单词卡

一个简单易用的单词记忆工具，支持从文本文件导入单词，具有单词卡片展示和听写训练功能。

# 立即试用
https://green798.github.io/Shorthand-Word-Card/

## 功能特点

- 📁 **文件导入**：支持从文本文件导入单词列表
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🔊 **语音朗读**：支持单词发音
- 📝 **听写训练**：可自定义间隔时间，支持无重复听写
- 🎨 **美观界面**：现代化卡片式设计

## 使用方法

### 1. 导入单词

1. 点击页面上方的"选择文件"按钮
2. 选择包含单词的文本文件（如 `单词(1).txt`）
3. 点击"导入单词"按钮
4. 系统会自动清空现有单词并导入新文件中的单词

### 2. 单词文件格式

单词文件应使用以下格式：

```
单词1	解释1
单词2	解释2
单词3	解释3
```

- 每行一个单词
- 单词和解释之间用制表符（Tab）分隔

### 3. 听写训练

1. 调整听写间隔时间（3-20秒）
2. 选择是否朗读单词解释
3. 点击"开始听写"按钮
4. 系统会随机播放单词，无重复
