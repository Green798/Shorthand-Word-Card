单词速记卡

2.0x
Mybatis
SSM框架之一，基于ORM思想的持久层框架，专门解决Java对象和数据库表的映射问题
Spring
SSM框架核心，Java开发的"春天"，核心是IoC/DI（控制反转/依赖注入），实现代码解耦
Spring MVC
SSM框架的Web层模块，基于MVC（模型-视图-控制器）架构，处理前端请求和响应
ORM
Object Relational Mapping（对象关系映射），将Java对象与数据库表建立映射关系
session
会话，Mybatis中SqlSession是与数据库的一次"对话"，负责执行SQL、获取连接
mapper
映射器，Mybatis核心组件，XML中的<mapper>标签用于编写SQL映射规则
properties
属性，Mybatis配置文件中用于加载外部属性文件（如db.properties），存储配置信息
settings
设置，Mybatis全局配置项（如日志、缓存），对应配置文件中的<settings>标签
alias
别名，Mybatis中为实体类设置简称，避免编写全类名，简化SQL编写
dataSource
数据源，数据库连接的"源头"，Mybatis通过它获取数据库连接
pool
池，通常指连接池（connection pool），管理数据库连接，避免频繁创建/销毁连接
